# SRTChatBot


Telegram 과 python SRTrain 모듈을 사용한 챗봇 프로그램 만들기

이 프로젝트는 챗봇 구현 프로젝트로 만들어 졌으며 상업적으로 사용 할 수 없습니다.

이 코드를 사용하여 발생하는 모든 법적 책임은 당사자에게 있으니 연구목적으로만 참고하시기 바랍니다.

Telegram > SRTChatBot 검색
1. 예약
2. 조회
3. 취소
4. 종료
